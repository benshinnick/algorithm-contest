# Sorting Algorithm Contest

https://benshinnick.github.io/algorithm-contest/

React application for visualizing sorting algorithms competing agianst each other in real time.

## Implemented Sorting Algorithms

* Merge
* Quick
* Heap
* Shell
* Insertion
* Selection
